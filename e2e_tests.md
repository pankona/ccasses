# e2e tests

agent-browser スキルを使ってブラウザ操作で検証する。
前提: `go run ./cmd/ccasses serve --port 8080` でサーバーが起動済みであること。

## トップページの初期表示

- http://localhost:8080 にアクセスする
- "Loading sessions..." が消え、セッション一覧が表示されることを確認する
- ヘッダーに "ccasses" と "Claude Code Session Assessment" というテキストが表示されていることを確認する
- 統計カード (Sessions, Total Turns, Output Tokens, Cache Read) が 4 つ表示され、それぞれ数値が入っていることを確認する

## チャート表示

- トップページにアクセスする
- "Tool Usage (All Sessions)" セクションが表示されていることを確認する
- "Model Usage (All Sessions)" セクションにドーナツチャート（canvas 要素）が描画されていることを確認する

## セッションテーブル

- トップページにアクセスする
- "Sessions" テーブルが表示されていることを確認する
- テーブルのヘッダーに ID, Project, Branch, Start, Duration, Turns, Output Tokens, Cache Read, Models の列があることを確認する
- テーブルに 1 行以上のデータ行が存在することを確認する

## セッション詳細への遷移

- トップページにアクセスする
- セッションテーブルの最初の行をクリックする
- "← Back to Sessions" ボタンが表示されることを確認する
- 詳細ヘッダーにプロジェクト名とセッション ID が表示されていることを確認する
- 統計カード (Turns, User Prompts, Output Tokens, Cache Read, Cache Created) が 5 つ表示されることを確認する

## セッション詳細のチャート

- セッション詳細画面に遷移する（テーブルの最初の行をクリック）
- "Tool Usage" セクションが表示されていることを確認する
- "Model Usage (turns)" セクションにドーナツチャートが表示されていることを確認する
- "Context Size Over Time" タイムラインチャートが表示されていることを確認する
- "Reset Zoom" ボタンが表示されていることを確認する

## セッション一覧への戻り

- セッション詳細画面に遷移する
- "← Back to Sessions" ボタンをクリックする
- セッション一覧画面に戻り、セッションテーブルが再び表示されることを確認する

## ツール使用のツリー展開

- トップページにアクセスする
- "Tool Usage (All Sessions)" セクション内で、▶ アイコンがついた行（子項目がある行）をクリックする
- 子項目が展開表示されることを確認する（▶ が ▼ に変わる）
- 同じ行を再度クリックして折りたたまれることを確認する

## URL ハッシュによる直接アクセス

- トップページにアクセスし、セッションテーブルの最初の行をクリックして詳細に遷移する
- URL に `#session/` が含まれていることを確認する
- そのURLをそのまま新しくアクセスして、詳細画面が直接表示されることを確認する

## エラー表示（API が応答しない場合）

- サーバーが起動していない状態で http://localhost:9999 にアクセスする
- ページがエラー状態になる（接続できない）ことを確認する
